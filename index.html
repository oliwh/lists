What is a list?

Lists are dynamic arrays, so the size of a list can be modified after runtime.

Lists are different to arrays in that they can store different data types. They store data contigiously.

List Operation Description Example Code
isEmpty() Test for empty List myList.isEmpty()
add(item) appends the specified item to the end of a list myList.add("item")
add(index, item) adds the specified item at the index specified myList.add(3, "hello")
clear() removes all elements in list myList.clear()
lastIndexOf(item) returns the index in the list of the last occurence of the element or -1 if it is not in the list myList.lastIndexOf(5)
remove(index) removes the element at the specified index myList.remove(4)
get(index) returns the element at the specified index myList.get(3)
size() returns the number of elements in the list myList.size()
contains(item) returns a boolean for whether the lists contains the specified item myList.contains(51)
indexOf(item) returns the index of the first occurence of the specified item in the list myList.indexOf(31)
